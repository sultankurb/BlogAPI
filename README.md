# This manifest to upgrade my skill in onion architecture
    This project will be a well-distributed monolith designed for high loads
    In this application, it is planned to become a basic service for a blog service in the near future,
    but in the future, when the basic functionality is  finalized, 
    new and complex features will be added until it reaches the level of a real project and will possibly be
    divided into microservices.

# Project dependencies
 * Docker
 * Docker Compose

# How to run app
## First step is clone this project 
    git clone https://github.com/sultankurb/BlogAPI.git

## The last step is to run
    cd BlogAPI/docker
    docker compose up -d
